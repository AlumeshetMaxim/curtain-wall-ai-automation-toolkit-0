# CW Facade Automation Toolkit - Extended Overview

This repository is a public open-source foundation for facade, curtain wall, CAD, PDF, DWG, Excel and STEP automation workflows.

## Main areas

- CAD layout and page setup workflows.
- PDF drawing package splitting and merging.
- DWG text and block extraction examples.
- MTO and BOM summary utilities.
- Revision comparison utilities.
- STEP annotation checking.
- Athena/profile workflow notes.

## Documentation

- `docs/script_catalog.md` - catalog of scripts.
- `docs/roadmap.md` - project roadmap.
- `docs/privacy_and_sanitization.md` - public repository safety rules.
- `docs/codex_oss_application.md` - notes for the Codex OSS application.

## Quick start

```bash
pip install -r requirements.txt
python scripts/example_mto_from_csv.py
python scripts/example_revision_compare.py
```

## Purpose

The goal is to make practical automation more accessible to facade planners, engineers and CAD users.
