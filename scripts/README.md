# Scripts

This folder contains public, sanitized workflow scripts and templates.

## Folders

- `cad/` - AutoCAD and BricsCAD workflow templates.
- `dwg/` - DWG extraction and quantity workflows.
- `pdf/` - PDF drawing-package tools.
- `excel/` - Excel schedule tools.
- `file_management/` - File replacement and organization tools.
- `checking/` - Validation and checking tools.
- `rhino/` - Rhino workflow scripts.
- `solidworks/` - SolidWorks workflow notes.
- `trimble/` - Trimble Connect workflow notes.
- `athena/` - Athena/profile workflow notes.

Keep private production files outside this repository.
